Prework Study Guide

HTML

  • is known as the structure of webpages
  • made of "elements"
  • text,image,headers, and lists
  • helps build structure and provides a sturdy framework
  • elements are defined by brackets
  • body defines elements for user
  • title defines a webpage title
  • meta defines character sets,keywords, author and viewpoints

CSS

  • there are 3 ways to style CSS
  • inline: apply to specific HTML tag, using style attributes for specific page
  • internal: CSS rules for webpage in the head section of HTML file
  • external: keep style changes in seperate file, seperation of concern links
  • created thorugh a selector, which has a property to change and a value assosiated with the change
  • asterisk is used for highlighting and doing all the symbols in a page

Git

  • use git status to check where you are in a directory
  • use git checkout to change between branches or to go create new branches in a directory
  • pull means to branch modifications on local environment
  • push means to put local repo changes into environment changes

Javascript

  • high level programming language that allows users to interact with websites
  • variables contain and store values
  • you can declare and assign values to variables and they are case senstive
  • variables include string, number, boolean, array, and object
  • additional values include conditionals, functions, and events
